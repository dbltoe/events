Greetings, I have (for some time) wanted to change my Scheduled Events mod to work strictly through the admin area of Zen Cart. I envision utilizing a similar format to the category_product_listing.php in admin/includes/modules. The mod would need to create the database and allow editing, copying, and additions from the category_product_listing.php very similar to the form for creating/editing a product. I envision all this being under the extras admin toolbar with a heading of Scheduled Events. Of course, it will need to be registered with the proper configuration areas in order to track its status. Once developed, Retail Outlets and Recipes will be soon to follow.
